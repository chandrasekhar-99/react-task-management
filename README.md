# Task Management System

A React.js application with Admin and User dashboards, supporting task management with drag-and-drop, task assignment, and completion tracking. All data is persisted in localStorage.


## Features
```
Admin

Create, assign, and manage tasks.

Drag-and-drop tasks to reassign users.

View tasks per user.

All tasks stored in localStorage.



User

View only tasks assigned to them.

Mark tasks as "Completed".

Task updates are saved to localStorage.

```


## Tech Stack
```

Frontend: React.js, React DnD, Tailwind CSS

State Management: React useState, useEffect

Storage: localStorage

Backend: None (Mocked with localStorage)

Deployment: Vercel

```

## Installation & Setup

### 1. Clone the repository

```

git clone https://github.com/your-username/task-management.git
cd task-management

```

### 2. Install dependencies

```

npm install

```

### Dependencies include:

```

npm install react-dnd react-dnd-html5-backend uuid react-router-dom

```

### 3. Start the development server

```

npm start

```

### 4. Open in browser

```

http://localhost:5173/

```
